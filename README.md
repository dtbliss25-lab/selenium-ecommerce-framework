# Selenium E-Commerce Automation Framework

## Tech Stack
This project uses the following technologies:

| Technology | Purpose |
|------------|---------|
| ☕ Java | Programming language used to develop automation scripts |
| 🌐 Selenium WebDriver | Browser automation framework for web application testing |
| 🧪 TestNG | Test framework for test execution, assertions, and test management |
| 📊 Allure Report | Test reporting framework for generating interactive test reports |
| ⚙️ GitHub Actions | CI/CD tool for automated test execution pipeline |
